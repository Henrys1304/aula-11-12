Glossário de Termos

HTML
HTML é a linguagem base usada para criar páginas da web. Ele estrutura o conteúdo de um site, dizendo ao navegador o que é um título, um parágrafo, uma imagem, um link, uma lista, uma tabela etc.

CSS
O CSS é usado para estilizar o HTML, ou seja, ele controla como o conteúdo de uma página web vai aparecer visualmente. Ele não cria conteúdo. Ele dá cor, forma, tamanho e layout ao conteúdo criado com HTML.
